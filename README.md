# E-commerce-Profissional
Aplicativo completo de e-commerce com gerenciamento de produtos, carrinho de compras, autenticação de usuários, pedidos e dashboard administrativo.
# E-commerce Profissional

## Tecnologias
- Frontend: React + TypeScript + Redux + TailwindCSS
- Backend: Node.js + Express + GraphQL + PostgreSQL + MongoDB
- Autenticação: JWT + OAuth
- DevOps: Docker + docker-compose
- Testes: Jest + Supertest

## Setup
1. Clonar repo
2. Rodar `docker-compose up --build`
3. Frontend: http://localhost:3000
4. Backend: http://localhost:5000/api

## Funcionalidades
- CRUD de produtos
- Carrinho de compras e checkout
- Autenticação de usuários (JWT)
- Dashboard administrativo
- Logs e analytics
- Deploy pronto para Kubernetes ou AWS
